# EyeMouse

EyeMouse is a project that uses a webcam to track eye movements and control the mouse pointer on your screen. It utilizes OpenCV, MediaPipe, and PyAutoGUI libraries.

## Requirements

- Python 3.x
- OpenCV
- MediaPipe
- PyAutoGUI

Usage
Ensure your webcam is connected.
Run the application using the command above.
The webcam feed will display, and your eye movements will control the mouse pointer.
Blink to simulate a mouse click.
